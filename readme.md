# Broadcast

ini adalah aplikasi web untuk broadcast jadwal pengajian dan jumatan melalui whatsapp

## Guide

1. Clone this repository
2. copy `.env.example` to `.env`
3. Install dependencies

```console
$ cd app && npm i
$ cd src/client && npm i
$ cd ../.. && npx prisma generate
```

4. Run the app

```console
$ npm run dev
```
